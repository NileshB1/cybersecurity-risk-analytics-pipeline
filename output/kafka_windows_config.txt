#### Kafka installation document on Windows:

1. Download latest version of kafka i.e. 4.2.0 (https://kafka.apache.org/community/downloads/)
2. Extract file (I kept in "D:" since windows path issues)
3. Open server.properties and add below contents (D:\kafka\config\server.properties):

#NileshB
process.roles=broker,controller
node.id=1

controller.quorum.voters=1@localhost:9093
controller.listener.names=CONTROLLER

listeners=PLAINTEXT://localhost:9092,CONTROLLER://localhost:9093
listener.security.protocol.map=CONTROLLER:PLAINTEXT,PLAINTEXT:PLAINTEXT

inter.broker.listener.name=PLAINTEXT

log.dirs=D:/kafka/logs

4. Format storage (One time):
